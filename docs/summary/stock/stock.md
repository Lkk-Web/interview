# 操作日志

记录每日操作与复盘，以及操作失误的教训总结。

## 交易原则

> 持续更新，约束自己的操作行为。

1. 不满仓操作（建议仓位 ≤70%）
2. 不听消息面盲目操作
3. 操作前先想好退出策略
4. **小盘股竞价技巧**：成交量<5000 万的股票，竞价可挂高 1-2 分钱
5. 可小幅做 t，操作不变形，缓慢收利，月目标做 t：仓位 1.5%；网格区间：1-2%
6. 保持现金流，不是：“怎么抄到底”，而是：“怎么保证自己一直活在市场里”
7. 单边下跌趋势，宁愿做空，也不要拿场资金外来做 T
8. 买入可挂支撑位的整数或 5 倍数位，资金通常喜欢保持这个数。比如 x.00,x.05。卖出
9. 永远保持自信，对自己的选择和操作自信，能避免 50%的操作失误
