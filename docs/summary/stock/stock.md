# 操作日志

## 交易原则

> 持续更新，约束自己的操作行为。

1. 不满仓操作（建议仓位≤70%）
2. 不听消息面盲目操作
3. 操作前先想好退出策略
4. **小盘股竞价技巧**：成交量<5000万的股票，竞价可挂高1-2分钱
5. 可小幅做t，操作不变形，缓慢收利，月目标做t：仓位 1.5%
6. 保持现金流，不是：“怎么抄到底”，而是：“怎么保证自己一直活在市场里”
7. 单边下跌趋势，宁愿做空，也不要拿场资金外来做T
